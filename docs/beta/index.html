What is Games HEL?

Games HEL is a one-day mini-conference in Helsinki focusing on Console and PC games. It's going to be a small, intimate event with ~100 attendees from the Finnish dev scene. We'll take over Games Factory for a day of talks, followed by evening mingling and finishing the night with an afterparty.
